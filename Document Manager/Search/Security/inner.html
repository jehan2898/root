The Web Document manager will allow users to launch a scan application executable to scan in paper documents into the system. This application will have features which will allow users to create tags/document types which will be displayed in a tree format.

The scanning application and Web document manager should interact in a way in that whatever tree structure is created in the scanning application should also appear in the document viewing application.
All tags/document types will be associated with a caseid/filenumber. Users will be able to view/save/annotate/print/fax/email the scanned documents. Also image processing functions like OCR, rotate, etc will be provided.
Data entry screen will be provided to enter index information if needed. Import functionality is needed to import soft copies of documents directly into the system.
